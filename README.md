
 Project Overview
This project models the relationship between tree height and diameter using a log-log regression model in R.

 Mathematical Model
ln(H) = β0 + β1 ln(D)

 Methodology
- Linear regression using lm()
- R² computation
- Statistical significance testing
- Graphical visualization

 Results
- High R² (~0.96) (bon ajustement!)
- Strong statistical significance
- Predictive model: H = exp(β0) * D^β1

 Tools
- R
- RStudio
